## Setup
### Clone 
```git clone https://github.com/abelzach/DFIR.git```
### Install Dependencies
```npm install```
### Start Front end Dapp
```npm run dev```

Visit localhost:3000 in your browser and use the Polygon Mumbai Test network in your Metamask wallet.
